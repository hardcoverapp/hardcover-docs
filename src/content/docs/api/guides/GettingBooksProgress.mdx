### Get all of the books you're currently reading.

<GraphQLExplorer query={`
query Me {
    me {
      id
      user_books(where: { status_id: { _eq: 2 } }) {
        id
        user_book_reads {
          progress_pages
        }
        book {
          id
          title
          pages
          image {
            url
          }
        }
      }
    }
  }
`} description='Get all books that you are currently reading' />
